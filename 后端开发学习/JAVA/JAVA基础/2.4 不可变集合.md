使用.of功能创建不可变集合
```java
List<String> list = List.of();

//创建集合后，用toArray变成数组，再塞进去
List<String> innerList = List.of("a", "b");
List<String> list = List.of(innerList.toArray(new String[0])); //展开集合元素
```
对于双键值的情况
```java
Map<String, Integer> scores = Map.of("Alice", 90, "Bob", 85);

Map<String, Integer> sourceMap = new HashMap<>();
sourceMap.put("A", 1);
sourceMap.put("B", 2);

//Map.of()最多只能接受10个键值对，而 Map.ofEntries()没有这个限制
Map<String, Integer> newMap = Map.ofEntries(
    sourceMap.entrySet().toArray(new Map.Entry[0]) 
);
```
